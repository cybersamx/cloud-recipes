# Shorthands of Commonly Used Kubernetes Objects

You can use the following shorthands in `kubectl`.

| Objects                 | Shorthands |
|-------------------------|------------|
| ConfigMap               | cm         |
| DaemonSet               | ds         |
| Deployment              | deploy     |
| HorizontalPodAutoScaler | hpa        |
| ingress                 | ing        |
| namespace               | ns         |
| node                    | no         |
| PersistentVolumeClaim   | pvc        |
| PersistentVolume        | pv         |
| Pod                     | po         |
| ReplicaSet              | rs         |
| ServiceAccount          | sa         |
| Service                 | svc        |
| StatefulSet             | sts        |
