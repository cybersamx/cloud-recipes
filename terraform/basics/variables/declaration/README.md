# Variables Declaration

Examples of how to declare variables of the most common used types in Terraform.

## Primitive Types

There are 3 primitive types in Terraform:

* bool
* number
* string

## Complex Types

Complex types exist in 2 primary forms in Terraform:

* object
* collection
  * list
  * map
  * tuple
